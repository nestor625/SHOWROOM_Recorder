# Showroom Recorder v1.0

GUI tool to record Japanese idol live streams from SHOWROOM.

## Files

- `SHOWROOM Recorder.bat` - Double-click to launch
- `showroom-recorder.ps1` - Main application
- `SHOWROOM_Recorder_教學.md` - Chinese tutorial
- `SHOWROOM_Recorder_Tutorial.md` - English tutorial

## Requirements

1. **Streamlink** - Install from Microsoft Store or https://streamlink.github.io/
2. **Python** - Required by Streamlink (check with `python --version`)

## Quick Start

1. Install Streamlink
2. Extract zip file
3. Double-click `SHOWROOM Recorder.bat`
4. Add channel URL + name
5. Click Record!

## Usage

See `SHOWROOM_Recorder_教學.md` for Chinese instructions.
See `SHOWROOM_Recorder_Tutorial.md` for English instructions.

## Recording Location

Default: `D:\chrome\`

Filename format: `NAME-SHOWROOM-YYYY-MM-DD_HH_MM.mp4`
