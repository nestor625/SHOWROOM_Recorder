# SHOWROOM Recorder Tutorial

## Installation
1. Install Streamlink: https://streamlink.github.io/
2. Extract SHOWROOM_Recorder_Win.zip
3. Double-click SHOWROOM Recorder.bat

## How to Use
1. Add channel URL (e.g., https://www.showroom-live.com/r/ME_HANA_OGI)
2. Add a name (e.g., Hana)
3. Click Add
4. Select channel and click Record

## Recording Location
Default: C:\Recordings\

Filename: NAME-SHOWROOM-YYYY-MM-DD_HH_MM.mp4
